## Recommended RoadMap:

Follow the Roadmap: 
```
1. Pre-requisite
2. Networking
3. Linux
4. Shell Scripting
5. Git
6. Python
7. Docker
8. Jenkins
9. AWS / Azure / GCP
10. Kubernetes
11. Terraform
12. Ansible
13. Prometheus
14. Grafana
15. Projects
16. Projects Documentation on GitHub
17. Resume and LinkedIn Profile Fixing
18. Starting Applying for Jobs
```
